### Projects

**[SGNL](https://github.com/metin-korkmaz/sgnl)** — Signal extraction engine that filters AI-generated slop from web content. Multi-layered scoring with heuristics, density analysis, and domain reputation. Built with FastAPI and Docker. Production-ready MVP.

**[QuotaLume](https://github.com/metin-korkmaz/quotalume)** — Minimal Linux tray app that monitors AI provider quotas (Codex, Claude Code, OpenRouter, Ollama Cloud) in real time. Written in Rust. No cloud, no stored credentials.

**[LunaGlow](https://github.com/metin-korkmaz/lunaglow-rp6)** — Android app for the Retroid Pocket 6 that drives ambient LED colors from on-screen content. Kotlin, Jetpack Compose, MediaProjection. 5 releases published.

**[RAG Lab](https://github.com/metin-korkmaz/rag-lab)** — Benchmarking notebook comparing sentence-transformers encoders (BGE, E5, MiniLM) and vector stores (FAISS, ChromaDB, Qdrant) on retrieval recall, MRR, and latency.
